# Global COVID-19 Dashboard in Power BI

## Overview
This project consists of a Power BI dashboard developed to analyze the evolution of COVID-19 cases worldwide using global data from Johns Hopkins University.

The dashboard was designed to explore confirmed cases, deaths, mortality rate, recovery rate, and daily case trends, with a special focus on Mexico for time-series analysis.

## Objective
The main objective of this project is to transform global COVID-19 data into an interactive visual dashboard that supports comparative analysis between countries and helps identify major pandemic patterns.

## Data Source
The dashboard is based on global COVID-19 reports provided by Johns Hopkins University, which served as one of the main international sources for pandemic tracking.

## Tools Used
- Power BI
- Power Query
- DAX
- Data modeling
- Interactive dashboard design

## Key Analysis Performed
The dashboard was developed to answer questions such as:

- Which are the 10 countries with the highest number of confirmed cases
- Which are the 10 countries with the highest number of deaths
- What is the global mortality rate
- Which countries have the highest mortality rate
- Which countries have the lowest mortality rate
- Which countries have the highest recovery rate
- How did daily COVID-19 cases evolve in Mexico over time

## Main Findings
According to the dashboard results:

- The 10 countries with the highest number of confirmed cases include US, India, France, Germany, Brazil, Japan, Korea South, Italy, United Kingdom, and Russia
- The 10 countries with the highest number of deaths include US, Brazil, India, Russia, Mexico, United Kingdom, Peru, Italy, Germany, and France
- The global mortality rate calculated in the dashboard is 1.02 percent
- Countries with the highest mortality rate include Korea North, MS Zaandam, Yemen, Sudan, Syria, Somalia, Peru, Egypt, Mexico, and Bosnia and Herzegovina
- Countries with the lowest mortality rate include Nauru, Bhutan, Burundi, Singapore, Tonga, Brunei, Marshall Islands, Korea South, New Zealand, and Vanuatu
- Countries with the highest recovery index include Diamond Princess, Holy See, Tajikistan, MS Zaandam, Djibouti, Kyrgyzstan, Colombia, Oman, India, and Nepal
- Mexico's daily cases curve shows several pandemic waves, with the strongest peak around early 2022, followed by later smaller rebounds

## Business and Analytical Value
This project demonstrates the analytical value of Power BI for transforming large and complex datasets into accessible visual insights. It allows users to compare countries, identify relative health impacts, and track temporal case behavior through interactive visuals.
